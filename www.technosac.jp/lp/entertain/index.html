学びの環境

“アニメの聖地”が学びの舞台。
最先端×安心感が得られる街！

  • # アニメの聖地
  • # 東京なのに安心

キャンパスがある東京・武蔵小金井は、人気の吉祥寺や三鷹などが並ぶJR中央本線沿いの駅。スタジオジブリが小金井市にあるのをはじめ、この中央線沿線には、ジブリにまつわるスポットが集まっています。新宿駅まで電車で約20分の武蔵小金井。駅周辺にも大型商業施設があり、買い物や食事に便利な一方、緑豊かで広大な「都立小金井公園」があり、ゆったりと過ごせます。東京の最先端に触れながら、地元のような安心感もある環境です。

国際交流

ニューヨークの姉妹校と交流！
本場のエンタメやアートを感じる。

  • # NYの大学と交流
  • # 日米でアニメ制作

姉妹校・ニューヨーク州立大学バーチェス校の学生と短期留学を通して交流を深めています。SDGsをテーマにしたアニメも共同で制作。時差や言葉・文化の壁を乗り越えながらオンラインミーティングを重ね、コロナ禍でも満足度の高いプログラムを実施。エンターテインメントやアートの本場・ニューヨークを肌で感じながら、クリエイティブの感性を高めています。

大学コース

経営、法律、文学なども学べる！
＋αの力で就職につなげる。

  • # 「学士」が取得できる
  • # Wメジャー制度

入学者の2.5人に1人が選択している大学コース。専門学校の学びが大学の卒業単位の約半分に認定され、大学卒業学位「学士」の取得が可能です。専門性に加えて、一般教養、経営学、語学などを学ぶことで就職につなげられます。また、大学コースを選択すると、2学科専攻（Wメジャー制度）も可能。例えば、ゲームクリエーター科と映像メディア学科の両方を選び、将来の活躍の幅を広げられます。奨学制度で限りなく学費の負担が少なく大学の科目を履修できます。

全学科横断ゼミにより
小金井市のPR動画を制作！

全学科でゼミが行われ、エンタメ・クリエイター系の学科同士のコラボによる実習や制作も多数。キャンパスのある小金井市の依頼により、市政施行60周年事業の一環として、市の魅力を世界へ発信するPR動画を学生たちが制作。脚本、撮影、音声、照明、編集、出演にまで「オールテクノス」で取り組みました。さらに、国分寺市からもPR動画制作の依頼があり制作中です。
