実践型学習

リアルな結婚式や観光ツアー…
本物にかかわりながら学ぶ！

  • # ウエディング・プロデュース
  • # オリジナルツアー

本物のカップルの結婚式を学生たちがプロデュース。企画提案、段取り、当日の運営をすべて担うプロジェクトがあり、さまざまな学科の学生が挑戦できます。他にも、学生がバスツアーの企画を競い、優勝者のツアーが実現するコンテストなどもあります。現場に飛び込み、本物にかかわる学びはテクノスならではの特色です。

業界との連携

名だたる企業とつながり
業界の最先端や一流に触れる！

  • # 現役ホテルパーソン
  • # 業界と太いパイプ

港区白金台に位置する伝統ある結婚式場・八芳園で模擬披露宴を実施。また、キャンパスで授業を担当する講師陣は、日系外資系ラグジュアリーホテルの現役ホテルパーソン。東京都内の一流ホテルでのフィールドワークの機会も多くあります。業界の名だたる企業とのつながりがあるからこそ可能な学びがあります。業界との太いパイプにより、帝国ホテル、ホテルオークラ東京、パークハイアット東京など都内ホテルを始め、全国のリゾートホテルへ、就職実績も抜群です。

大学コース

「大学コース」で＋αの力を。
経営学や心理学を学ぶ。

  • # 「学士」が取得できる
  • # Wメジャー制度

大学で経営学を学んでホテルの総支配人を目指す。心理学を接客に活かす。海外での活躍を見据えて語学を学ぶ。大学コースを選択することで、将来の幅を広げられます。入学者の2.5人に1人が選択している大学コースは、奨学制度で限りなく学費の負担が少なく大学の科目を履修できます。専門学校の学びが大学の卒業単位の約半分に認定され、大学卒業学位「学士」を取得できます。また、大学コースでは2学科専攻（Wメジャー制度）も可能です。

今、世の中で話題の
ツアー商品を学生たちが企画。

長野県佐久市にあるテクノス所有の温泉宿泊施設「みどりの村」。建築、情報システム、電気電子の各学科の学生たちがコラボし、自然エネルギーを利用したスマートビレッジづくりに挑戦。１年次の研修を始め、さまざまな学科のゼミ活動や資格対策合宿で利用しているこの施設を舞台に、観光・ツーリズム科の学生たちは、観光地で仕事をするワーケーションのツアー商品を企画しています。
